# Design System Standards

## Card Component
Standardized card design used across all view pages (Pantry, Wallet, Assets, etc.)

### CSS Classes
```
bg-bg-card p-4 rounded-2xl border border-border-dark shadow-sm
```

### Specifications
- **Background**: `bg-bg-card` (Dark grey/Deep theme background)
- **Padding**: `p-4` (Standard proportional padding)
- **Corner Radius**: `rounded-2xl` (Smooth modern rounded corners)
- **Border**: `border border-border-dark` (Subtle separation)
- **Effect**: `shadow-sm` (Light depth)

## Hierarchy
- **Header**: Use `font-black text-white uppercase tracking-tight` for titles within cards.
- **Subtext/Labels**: Use `text-[9px] font-black uppercase tracking-[0.15em] text-text-muted` for metadata labels.
